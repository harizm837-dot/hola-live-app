## Architecture Principles

- Modular file structure
- Expandable without breaking
- Clear separation:
  - Core logic
  - UI components
  - Economy engine
  - Live engine
- localStorage based persistence (Phase 1)
- Service Worker enabled
- GitHub Pages compatible
